<h1 align="center">Цель и задачи работы</h1>
<hr>
<p><b>Цель:</b> Разработка программно-аппаратного комплекса (ПАК) независимого модуля системы наведения на объект для космического аппарата форм-фактора CubeSat 1U.</p>
<p><b>Задачи:</b></p>
<ol>
  <li>Спроектировать механическую конструкцию модуля, совместимую с креплениями и размерами стандарта CubeSat 1U (100×100×100 мм).</li>
  <li>Реализовать систему автоматического наведения лазерного целеуказателя по двум взаимно перпендикулярным осям (наклона и поворота).</li>
  <li>Обеспечить угол обзора не менее 80° (±40° от центра) по каждой оси с шагом наведения 10°.</li>
  <li>Разработать систему радиосвязи (прием управляющих сигналов и передача телеметрии).</li>
  <li>Реализовать программный алгоритм, выполняющий тестовые циклы сканирования (горизонтальный, вертикальный, диагональные) и обработку команд.</li>
</ol>

<h2 align="center">Модуль обеспечивает:</h2>
<hr>
<ul>
  <li><b>Прецизионное управление лучом:</b> Двухосевое позиционирование лазера в диапазоне ±40° (шаг 10°), исходящего из геометрического центра корпуса CubeSat 1U;</li>
  <li><b>Дистанционное взаимодействие:</b> Поддержка беспроводного протокола управления через радиомодуль NRF24L01+ PA + LNA;</li>
  <li><b>Диагностические алгоритмы:</b> Автоматический цикл проверки, включающий сканирование по горизонтали, вертикали и двум диагональным направлениям;</li>
  <li><b>Быстрый старт:</b> Переход исполнительной части в активный режим через 3 секунду после подачи питания;</li>
  <li><b>Сбор данных:</b> Передача пакетов телеметрии с указанием уникального ID устройства, актуальных координат наведения и статуса текущего режима.</li>
</ul>

<h2 align="center">Техническое оснащение системы:</h2>
<hr>
<ul>
  <li><b>Arduino Nano</b> — центральный узел наземного сегмента, транслирующий управляющие координаты после получения стартового сигнала с компьютера;</li>
  <li><b>Arduino Nano</b> — бортовой вычислитель CubeSat, отвечающий за работу лазерной головки KY-008 и приводных механизмов SG90;</li>
  <li><b>Радиомодуль NRF24L01+ PA + LNA</b> — интерфейс для передачи данных в одностороннем режиме от наземной станции к CubeSat;</li>
  <li><b>3D-печатное шасси</b> — оболочка форм-фактора 1U, спроектированная для фиксации лазера строго в центральной точке конструкции.</li>
</ul>
